Logical Equivalences

Equivalence Name p ∧ ⊤ ≡ p p ∨ ⊥ ≡ p Identity laws p ∨ ⊤ ≡ ⊤   p ∧ ⊥ ≡ ⊥ Domination laws p ∨ p ≡ p   p ∧ p ≡ p Idempotent or tautology laws ¬ ( ¬ p ) ≡ p Double negation law p ∨ q ≡ q ∨ p   p ∧ q ≡ q ∧ p Commutative laws ( p ∨ q ) ∨ r ≡ p ∨ ( q ∨ r )   ( p ∧ q ) ∧ r ≡ p ∧ ( q ∧ r ) Associative laws p ∨ ( q ∧ r ) ≡ ( p ∨ q ) ∧ ( p ∨ r )   p ∧ ( q ∨ r ) ≡ ( p ∧ q ) ∨ ( p ∧ r ) Distributive laws ¬ ( p ∧ q ) ≡ ¬ p ∨ ¬ q   ¬ ( p ∨ q ) ≡ ¬ p ∧ ¬ q De Morgan's laws p ∨ ( p ∧ q ) ≡ p   p ∧ ( p ∨ q ) ≡ p Absorption laws p ∨ ¬ p ≡ ⊤   p ∧ ¬ p ≡ ⊥ Negation laws

Logical Equivalences Involving Conditional Statements

p → q ≡ ¬ p ∨ q   p → q ≡ ¬ q → ¬ p   p ∨ q ≡ ¬ p → q   p ∧ q ≡ ¬ ( p → ¬ q )   ¬ ( p → q ) ≡ p ∧ ¬ q   ( p → q ) ∧ ( p → r ) ≡ p → ( q ∧ r )   ( p → q ) ∨ ( p → r ) ≡ p → ( q ∨ r )   ( p → r ) ∧ ( q → r ) ≡ ( p ∨ q ) → r   ( p → r ) ∨ ( q → r ) ≡ ( p ∧ q ) → r

Logical Equivalences Involving Biconditional Statements

p ↔ q ≡ ( p → q ) ∧ ( q → p )   p ↔ q ≡ ¬ p ↔ ¬ q   p ↔ q ≡ ( p ∧ q ) ∨ ( ¬ p ∧ ¬ q )   ¬ ( p ↔ q ) ≡ p ↔ ¬ q
